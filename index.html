Variáveis

 Posted on September 20, 2017
As variáveis em PHP podem ser definidas com umas infinidade de nomes, porém, mesmo com toda a flexibilidade que a linguagem oferece existem algumas regrinhas e tipos para serem observados. Regra: Toda variável em PHP deve iniciar com $ (cifrão). Em seguida deve vir um caracter não numérico, isso inclui _ e -. A gama de definições possível para variáveis é representada pela seguinte expressão regular: [a-zA-Z\x7f-\xff][a-zA-Z0-9\x7f-\xff]* Tipos Apesar de não ser uma linguagem fortemente tipada, o PHP trabalha com alguns tipos primitivos, comuns entre outros lingagens, tipos compostos e especiais.
